    AlchemicalOpenMMSimulation¶

    class absolv.simulations.AlchemicalOpenMMSimulation(system, coordinates, box_vectors, state, protocol, lambda_index, platform)[source]¶

    An extension to the base equilibrium simulation class that will additionally compute the reduced potentials at each lambda value after each production simulation iteration and save those values to disk.
